Connexion à Supabase

Supabase est une base de données gratuite en ligne. Vos données seront sauvegardées et accessibles depuis n'importe quel appareil.

📋 Instructions (5 minutes) 1. Créez un compte gratuit sur supabase.com
2. Créez un nouveau projet (notez votre région → choisissez EU West)
3. Dans SQL Editor, exécutez cette requête :
CREATE TABLE injurytrack ( id integer PRIMARY KEY, data jsonb NOT NULL, updated_at timestamptz DEFAULT now() ); ALTER TABLE injurytrack ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_access" ON injurytrack FOR ALL USING (true) WITH CHECK (true); -- Active la synchronisation temps réel ALTER PUBLICATION supabase_realtime ADD TABLE injurytrack; 4. Dans Settings → API, copiez l'URL du projet et la clé anon/public
